    DimensionalData.jl

    Julia Datasets with named dimensions

    High performance name indexing for Julia

    Getting Started
    View on Github
    API reference

    Intelligent Indexing

    It works with datasets that have named dimensions. It provides no-cost abstractions for named indexing, and fast index lookups.

    Powerful Array Manipulation

    Permutedims, reduce, mapreduce, adjoint, transpose, broadcasting etc., and groupby operations.

    Seamlessly integrated with the julia ecosystem

    Works with base methods. If a method accepts numeric indices or dims=X in base, you should be able to use DimensionalData.jl dims.
